# programmers Lv1. MySQL 하루 2문제(24.09.25)

## 1) 아픈 동물 찾기

- 아픈 동물의 아이디와 이름을 조회

```sql
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION = "Sick";
```

## 2) 어린 동물 찾기

- 동물 중 젊은 동물의 아이디와 이름을 조회

```sql
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION != 'Aged'
```
